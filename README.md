# AgriSmart 🌾

**AgriSmart** is a comprehensive, Android-based smart farming solution designed to bridge the gap between traditional farming and modern technology. Developed using **Java** and **Firebase**, it empowers farmers with real-time data, AI-driven insights, and easy access to essential agricultural resources.

## 🚀 Key Features

*   **Real-time Market Prices:** Stay updated with the latest crop prices from local and national markets (Mandis) to make informed selling decisions.
*   **AI-based Crop Suggestions:** Input soil parameters and weather conditions to receive intelligent recommendations on the most profitable crops to plant.
*   **Nearby Agricultural Stores:** Locate the nearest seed, fertilizer, and equipment suppliers using integrated Maps and location services.
*   **Government Schemes:** A curated database of the latest government subsidies and schemes tailored to help farmers maximize their benefits.
*   **Farming Resources:** Access a library of best practices, modern techniques, and educational content for sustainable farming.
